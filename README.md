#  DevOps Lifecycle Dashboard & Digital Identity Portfolio

A production-ready Django project that combines a **cybersecurity portfolio** with a **DevOps monitoring & control dashboard**.

This project demonstrates a full **DevOps lifecycle** including containerization, deployment, monitoring, and CI/CD automation.

---

##  Key Features

- 🔥 DevOps Dashboard (CPU, RAM, Docker containers)
- ⚙️ Container Control (Start / Stop / Restart)
- 📜 Logs Viewer (basic container logs)
- 🐳 Dockerized Application (Docker + docker-compose)
- 🔄 CI/CD Pipeline (GitHub Actions)
- 🌐 Dynamic Portfolio (Django + Admin panel)

---

##  Tech Stack

- **Backend:** Django
- **DevOps:** Docker, Docker Compose
- **CI/CD:** GitHub Actions
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite (dev) / PostgreSQL (production-ready)

---

## 🔄 DevOps Lifecycle Covered

This project simulates a real-world DevOps workflow




- Build: Django application
- Package: Docker image
- Deploy: docker-compose
- Run: containers
- Monitor: CPU, RAM, container status
- Control: restart / stop / start containers
