🚀 QUICK START (COPY & PASTE)

# Navigate to project
cd cyber_portfolio

# Create virtual environment
python -m venv venv

# Activate it (Windows)
venv\Scripts\activate
# OR (macOS/Linux)
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Setup database
python manage.py migrate

# Create admin account
python manage.py createsuperuser
# (Enter username, email, password when prompted)

# Run server
python manage.py runserver

# Access the site
# Portfolio:    http://127.0.0.1:8000/
# Admin Panel:  http://127.0.0.1:8000/admin/

🚨 IMPORTANT NOTES
═══════════════════════════════════════════════════════════════════════════════

⚠️ BEFORE PRODUCTION
   1. Change SECRET_KEY in settings.py (run: python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())")
   2. Set DEBUG = False
   3. Update ALLOWED_HOSTS with your domain
   4. Enable HTTPS/SSL certificate
   5. Switch to PostgreSQL
   6. Configure email backend
   7. Set up regular backups

⚠️ DURING DEVELOPMENT
   1. Keep DEBUG = True for error details
   2. Use SQLite for easy testing
   3. Use Django admin to manage content
   4. Test on mobile (responsive)
   5. Check console for JavaScript errors (F12)

⚠️ COMMON ISSUES
   • Static files not loading? Run: python manage.py collectstatic
   • Database errors? Run: python manage.py migrate
   • Admin not working? Create superuser: python manage.py createsuperuser
   • Port 8000 in use? Try: python manage.py runserver 8001
